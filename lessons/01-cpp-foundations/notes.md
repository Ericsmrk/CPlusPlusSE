# Lesson 1: C++ Foundations + Encapsulation

## C++ focus
- program structure
- `main()`
- headers and namespaces
- variables and types
- input/output
- basic functions

## Software engineering focus
Even simple programs benefit from structure. Early on, we want to build the habit of:
- naming things clearly
- separating responsibilities
- keeping functions small
- avoiding duplicated logic

## Example principle
A function that both asks the user for input, validates it, stores it, and prints a report is doing too much. Breaking that work apart is an early form of good modularity.

## Key idea
Learning C++ and software engineering together means asking two questions:
1. Does the code work?
2. Is the code organized in a way that will still make sense later?
