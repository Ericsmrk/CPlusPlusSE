# Lesson 1: C++ Foundations

This folder contains the first lesson materials: **program structure**, **basic I/O**, **simple functions**, and light habits for readable code.

| File | Purpose |
|------|---------|
| `notes.md` | Full lesson: goals, concepts, annotated examples, build/run, common mistakes |
| `exercises.md` | Hands-on tasks |
| `quiz.md` | Short check with answer key |
| `src/` | Runnable examples wired in the root `CMakeLists.txt` |

**Primary C++ focus:** `#include`, `using namespace std;`, `main()`, `cout` / `cin` / `string`, small functions, `const` at an introductory level.

**Software engineering (light):** clear naming, short `main()`, early decomposition—no patterns or heavy theory in this lesson.
